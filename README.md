<a id="-instalação"></a>

## ⚙️ Instalações


### FRONT-END | ViteJs - ReactJs - TypeScript

Cria um projeto com ViteJs

```bash
npm create vite@latest
# Project name: » dt-money | Select a framework: » React | Select a variant: » TypeScript
npm install
```

### Executando o aplicativo web

```bash
cd dt-money
npm run dev
```
